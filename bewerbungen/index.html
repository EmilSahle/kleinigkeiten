What I know

An der Uni Köln habe ich zuletzt das "IT-Zertifikat" absolviert. Dazu habe ich vier Kurse besucht: "Advanced IT Basics", "Advanced Web Basics", "Tools and Methods in DH" und "Digital Objects".

Als eine Kursabgabe habe ich eine Website entwickelt, die Daten zu Geographie-Studiengängen (von mir zusammengestellt und in einem JSON-File abgelegt) auf einer Leaflet-Karte visualisiert. Mein Java-Script sorgt dann dafür, dass Checkbox-Auswahlen zu einer dynamischen Veränderung der Visualisierung führen. Daten und Scripte liegen in einem GitHub-Repo, die Anwendung läuft über GitHub Pages.

Check it out!

How would I know?!

Ich kann also ein kleines bisschen Web-Tech (HTML, CSS, JS). Aber ich bin kein Programmierer. Was würde ich als nächstes lernen? Programmiersprachen? Python, PHP? Web-Frameworks? Angular, React, Framer 😉? Bis zum Ausbildungsbeginn ist noch etwas Zeit. Ich könnte ja schon mal ein paar Onlinekurse machen ...

Was habe ich getan?! 😱

Meine Bewerbung als Klon eurer Webseite. Schöne Idee, oder? Habe ich auch erst gedacht. Code kopieren und Inhalte ändern kann ja nicht so schwer sein. Habe ich gedacht. Aber die Webseite ist ja das Produkt des Website Builder "Framer". Und der erzeugt einen völlig unverständlichen Code. Es war eine echte Qual. Jetzt ist er noch schlimmer ... . So mache ich das nie wieder.

Mir ist völlig klar, dass das hier copyright-mäßig höchst fragwürdig ist. Deshalb verschwindet die Website sofort aus dem Netzt, wenn ihr sie gesehen habt. Sagt einfach Bescheid.
